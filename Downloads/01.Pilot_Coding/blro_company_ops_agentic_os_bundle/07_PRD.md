# PRD — BLRO Company Operating OS

## 1. 제품명
BLRO Company Operating OS

## 2. 한 줄 설명
메일, 고객, 프로젝트, 견적, 기술검토, 일정, 비용을 하나로 묶어 대표가 오늘 해야 할 일을 자동으로 판단하고 실행 초안까지 만드는 AI 운영 시스템.

## 3. 문제 정의
베를로 운영은 고객 메일, 파트너 요청, 기술 검토, 견적, 프로젝트 진행, 비용 관리가 동시에 발생한다. 현재 구조에서는 중요한 요청이 메일/회의/메신저/문서에 흩어져 있고, 대표가 직접 기억하고 정리해야 한다.

## 4. 목표
- 오늘 처리해야 할 업무를 자동 브리핑한다.
- 고객 메일을 분석해 요청사항과 다음 액션을 추출한다.
- 고객 요청을 프로젝트로 연결한다.
- 프리세일즈 기술검토 체크리스트를 자동 생성한다.
- 견적/제안 초안을 만든다.
- 외부 발송과 견적 발송은 승인 후 처리한다.
- 모든 실행 결과를 로그로 남긴다.

## 5. 주요 사용자
| 페르소나 | 목적 |
|---|---|
| 대표 | 회사 운영 현황과 우선순위 판단 |
| 영업 | 고객별 제안 기회와 후속 액션 관리 |
| 프리세일즈 | 기술 문의 검토와 회신 초안 |
| 기술지원 | PoC/장애/작업내역 정리 |
| 운영 PM | 프로젝트 상태와 지연 업무 관리 |
| 재무 | 매출/비용/수익성 관리 |
| 고객 | 명확한 요청자료와 답변 수신 |
| 파트너 | 정리된 요구사항과 견적 요청 수신 |

## 6. 핵심 기능
운영 브리핑, 메일 인텔리전스, 고객/프로젝트 관리, 프리세일즈 검토, 견적/제안, 승인/로그, 재무 관리.

## 7. 성공 지표
| 지표 | 목표 |
|---|---|
| 첫 업무 브리핑 확인 시간 | 3분 이하 |
| 메일 분류 정확도 | 80% 이상 |
| 첫 답장 초안 생성 시간 | 1분 이하 |
| 승인 정책 적용률 | 100% |
| 기술검토 체크리스트 생성률 | 90% 이상 |

## 8. MVP 범위
### P0
Dashboard, Customers, Projects, Mail Intelligence, Presales Review, Agents, Command Center, Approvals, Runs/Logs

### P1
Proposal/Quote Desk, Workflows, Finance, Settings, Tool Connections

| 견적 수치 오류 | 최종 금액은 수동 확인/승인 |
